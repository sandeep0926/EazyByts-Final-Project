## Setup Instructions
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/krupaliChotaliya/EazyByts_FitHub.git
   ```
2. **Navigate to the Project Directory**:
   ```bash
   cd EazyByts_FitHub
   ```
3. **Build the Project**:
   ```bash
   mvn clean install
   ```
4. **Run the Application**:
   ```bash
   mvn spring-boot:run
   ```
